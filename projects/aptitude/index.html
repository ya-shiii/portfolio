// OVERVIEW

Aptitude is a custom event management and ticketing platform designed for an entertainment business. The system brings event publishing, ticket purchasing workflows, attendee information, and administrative tools into a single platform while supporting payment and analytics integrations.

// THE PROBLEM

The business needed a dedicated platform for managing events and online ticketing instead of relying entirely on disconnected third-party tools and manual administrative workflows.

// THE SOLUTION

Designed and developed a Nuxt.js frontend backed by Django REST Framework and PostgreSQL. The platform supports event management, ticket purchasing workflows, administrative operations, and Stripe and PayMongo integrations.
